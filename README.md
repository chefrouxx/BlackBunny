# BlackBunny

BlackBunny is a modern social networking app from Black Bunny Game Studios. It connects friends with real-time chat, post sharing, groups, and friendly social discovery. Built with Expo, React Native, Firebase, and Electron, it supports mobile, web, and desktop platforms.

## 🚀 What BlackBunny Does

- Real-time chat and messaging
- Social feed with posts and reactions
- Friend requests and group support
- Offline support with local storage
- Cross-platform experience: iOS, Android, Web, macOS, Windows
- Constant updates and improvements from Black Bunny Game Studios

## 📥 Download the App

### macOS
1. Open the latest GitHub release page:
   `https://github.com/chefrouxx/BlackBunny/releases`
2. Download the latest macOS installer or zip file
3. Open it and follow the install instructions
4. Launch `BlackBunny` from your Applications folder

### Windows
1. Open the latest GitHub release page:
   `https://github.com/chefrouxx/BlackBunny/releases`
2. Download the latest Windows installer:
   - `BlackBunny-Setup.exe`
   - or `BlackBunny-1.0.0.exe`
3. Run the installer
4. Launch BlackBunny from the Start menu or desktop shortcut

### iOS / Android
- For iOS: install Expo Go and open the published Expo link
- For Android: install Expo Go and open the published Expo link
- For web: use the published web app URL if available

> Black Bunny Game Studios updates BlackBunny regularly with new features, bug fixes, and performance improvements. Check GitHub Releases often for the latest desktop installers and release notes.
